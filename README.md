# 🚀 Ship Relight Guardian

> Fault-tolerant propulsion decision system for engine-out detection, isolation, and autonomous relight authorization in multi-engine aerospace architectures.

---

## 📡 System Overview

Ship Relight Guardian simulates a redundancy-based propulsion control architecture designed for high-reliability engine management.

The system continuously evaluates engine health, isolates failures, and determines safe relight eligibility under constrained flight conditions.
